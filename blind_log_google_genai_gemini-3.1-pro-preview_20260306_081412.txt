### 2. The Electrum Standard (Version Numbering)
Electrum's creator, ThomasV, decided against using BIP39. Instead, Electrum uses a system where the **version number and wallet type are mathematically baked into the phrase itself.**

*   If the hash starts with `01`, the wallet knows it's a Legacy wallet.
*   If the hash starts with `100`, the wallet knows it's a native SegWit wallet.

If you type 12 random words into Electrum, the hash likely won't start with either of those specific prefixes, and Electrum will reject it as an invalid seed.
